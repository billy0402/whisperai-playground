# python-template

My Python template.

## Development environment

- [macOS 12.7.6](https://www.apple.com/tw/macos/monterey/)
- [Visual Studio Code 1.93.0](https://code.visualstudio.com/)
- [Python 3.11.7](https://www.python.org/)

## Installation

```shell
$ make install
```

## Getting Started

```shell
$ python src/main.py
```

## Lint

```shell
# style checking
$ make lint

# type checking
$ make typecheck
```

## Testing

```shell
$ make test
```
